# Atlas 设计系统

> 讲清 5 套主题的设计哲学、配色纪律、质感处理、艺术层。适合讲设计思考或定制主题。

## 1. 设计哲学

Atlas 的视觉目标:**有设计感,但不花哨;舒适耐看,支持长时间阅读写作**。

三条铁律:
1. **每套主题有清晰的设计哲学**——不是随便配色,是"这个主题想表达什么"
2. **一个主调,饱和度 < 80%**——不荧光、不刺眼,长时间看不累
3. **色温在主题内统一**——要么全暖,要么全冷,不混

## 2. 主题双维度架构

```html
<html data-theme="nocturne" data-mode="dark">
```

- **调色板**(`data-theme`):ink / parchment / obsidian / mist / nocturne
- **浅深**(`data-mode`):light / dark / 不设(跟随系统)

两个维度独立。5 套主题各有 light/dark 双形态(obsidian 主推 dark,parchment 主推 light,各自另一态有兼容兜底)。

实现:每套主题在 `themes.css` 里用 `:root[data-theme="xxx"]` 重定义全套 token。

## 4. 质感处理(全主题通用)

### 暖纸光晕
body 背景三层叠加:
```css
background:
  radial-gradient(... at 88% -10%, 强调色微光),   /* 右上 */
  radial-gradient(... at 8% 105%, 辅色微光),      /* 左下 */
  var(--bg);
```
让背景不死板,有"光从角落来"的感觉。

### 噪点纹理
`body::before` 用 inline SVG fractalNoise,`mix-blend-mode: multiply`,opacity 0.5。模拟纸张/材质的颗粒感。

### glass 效果
顶栏、模态遮罩用 `backdrop-filter: blur(16-20px) saturate(1.4-1.6)` + 半透明白 + inset 高光边。元素浮在内容之上,有层次。

### 阴影分层
```
--shadow-xs/sm/lg/xl 四档 + --shadow-inset(顶高光)
```
柔光扩散(不是硬阴影),Bento 风格。深色模式阴影更重(透明度更高)才有立体感。

## 5. 对比度纪律

暖色调容易"软"。Atlas 的处理:
- **三档面板明度差**:bg(暗)→ panel-soft(侧栏,中)→ panel(主区,亮),层次分明
- **边框够深**:`--border` 不用接近背景的浅色,确保卡片边界清晰
- **文字对比度 ≥ 7:1**(主文字)/ ≥ 4.5:1(次文字),长时间阅读不累
- **`--on-accent`**:彩色背景上的白字用独立 token(深浅模式都白),避免深色模式文字消失

## 6. 动效纪律

- **GPU-only**:只动 `transform` / `opacity` / `filter` / `clip-path`,绝不动 width/height/margin(避免重排)
- **缓动变量**:`--ease-out`(平滑减速)/ `--ease-spring`(弹性)/ `--ease-in-out`
- **触感反馈**:按钮按下 `scale(0.97-0.98)`,卡片悬浮 `translateY(-2px)`
- **永续微动效**:状态点呼吸、toast 弹性进入、烛光摇曳(夜曲)
- **`prefers-reduced-motion`**:全局降级,尊重无障碍

## 7. 如何加一套新主题

1. 在 `themes.css` 加 `:root[data-theme="yourname"]` 块,重定义全套 token(配色 + 状态色 + code-bg)
2. 想要专属背景/装饰,加 `:root[data-theme="yourname"] body { background: ... }`
3. 在 `lib/theme.ts` 的 `THEMES` 数组加元数据(name/tagline/swatch 色卡)
4. ThemePicker 自动出现新主题选项

**约束**:必须遵守三档面板分层 + 对比度纪律,否则会"软"/"糊"。
